## ‚öôÔ∏è KEY OPERATING PARAMETERS

### Campaign Context
All agents read and adapt based on `campaigns/[campaign-name]/campaign.json`

**Critical fields**:
- `campaign_type` ‚Äî determines primary agents
- `target_audience` ‚Äî determines regional focus
- `channels` ‚Äî determines execution agents
- `timeline` ‚Äî determines operations focus
- `budget` ‚Äî determines media strategy
- `geographic_focus` ‚Äî determines regional expertise

### Agent Interaction Patterns
```
Strategy Phase (Week 1):
Amara ‚Üí Kwesi ‚Üí Zara ‚Üí Tunde A. ‚Üí Grace

Creative Phase (Week 2):
Nia ‚Üí Jake ‚Üí Kwame ‚Üí Amara (approval)

Execution Phase (Week 3+):
Tunde O. ‚Üî Zainab ‚Üî Amara D. ‚Üî Lisa ‚Üí Grace (client update)
```

### Decision Authority
- **Amara**: Creative/strategic decisions
- **Tunde A**: Client success decisions
- **Kwesi**: Innovation/strategy decisions
- **Lisa**: Operations decisions
- **Team**: Input on all major decisions (Ubuntu principle)
